Aggies Compute

This project was designed to assist students with computer-science-related math, similar to applications such as Wolfram Alpha.

With the help of like-minded programmers, we created multiple math algorithms and implemented them into a website.

This collaborative project helped me develop problem-solving, teamwork and introduced me to new coding concepts and languages.

Language(s) used: React, Tailwind/CSS(style), Typescript, Apollo-client (Connection)

Link: Aggie Coding Club

Image Scaling / Image Stitching Algorithm

During this project, I created algorithms to scale individual images and stitch two images together. In this project, I bolstered my coding skill in C++ and further understood concepts such as 2D array manipulation, algorithms, and input/output streams. I expanded upon my knowledge of bilinear interpolation to implement the image scaling algorithm. Throughout this experience, I also gained an understanding of Harris's corner detection algorithm for the creation of the image stitching algorithm.

Language(s) used: C++

Link: Harris Corner Detection Algorithm

Dungeon Crawler Game

In this project I developed a retro-style video game using concepts such as pointers and 2D dynamic arrays.

During the process of creating this video game, I sharpened my understanding of pointers, input/output streams,

file traversal, memory allocataion and resizing/editing 2D dynamic arrays without risk of memory leaks.

This project also helped me learn the importance of creating test/edge cases, as I created my own levels that tested

the accuracy, precision, and stability of the code.

Language(s) used: C++

Custom Language Interpreter for "MUPL"

This project is centered around "MUPL", a Made Up Programming Language (hence the name MUPL) that has the basic functionality of common coding languages. I created a functional, statically typed language with functions, closures, and many different data types. Additionally, I made error handling, function calling, recursion, and let expressions all compatible for this custom language all using Racket as a foundation for my language. Due to my deep understanding of programming language concepts, I created hall of fame functions and complex algorithms to test my "MUPL" language.

Language(s) used: Racket, "MUPL"

Tetris

Using Ruby, I created the puzzle video game Tetris with a simple interface, buttons, and a score counter.

While working on this project, I developed my knowledge of Object Oriented Programming by working with classes,

subclasses, and dynamic dispatch.

Language(s) used: Ruby

Link: Tetris
